convection_diffusion_stabilized
A 1D convection-diffusion problem

convection_diffusion_stabilized, a FENICS script which uses the finite element method to simulate the solution of a 1D convection-diffusion problem, using artificial viscosity to stabilize the solution.

Note that I have installed FENICS using Docker, and so to run this script I issue the commands:

  1. cd $HOME/fenicsproject/convection_diffusion_stabilized
  2. fenicsproject run
  3. python3 convection_diffusion_stabilized.py
  4. exit
