domain
Generate a Region by "Subtraction"

domain, a FENICS script which which creates a region starting with a circle, and then "subtracting" a rectangle and smaller circle.

Note that I have installed FENICS using Docker, and so to run this script I issue the commands:

  1. cd $HOME/fenicsproject/domain
  2. fenicsproject run
  3. python3 domain.py
  4. exit
