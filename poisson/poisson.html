poisson
FENICS Solution of Poisson Equation

poisson, a FENICS script which uses the finite element method to solve a version of the Poisson equation over the unit square, adapted from the FENICS tutorial by Langtangen and Logg.

Note that I have installed FENICS using Docker, and so to run this script I issue the commands:

  1. cd $HOME/fenicsproject/poisson
  2. fenicsproject run
  3. python3 poisson.py
  4. exit
