dpg_bvp
Discontinuous Petrov Galerkin for a Boundary Value Problem

dpg_bvp, a FENICS script which uses the Discontinuous Petrov Galerkin (DPG) method to solve a boundary value problem over the unit interval, by Jay Gopalakrishnan.

Note that I have installed FENICS using Docker, and so to run this script I issue the commands:

  1. cd $HOME/fenicsproject/dpg_bvp
  2. fenicsproject run
  3. python3 dpg_bvp.py
  4. exit
