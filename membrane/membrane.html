membrane
Deflection of a Circular Elastic Membrane

membrane, a FENICS script which models the deflection of a circular elastic membrane subject to a perpendicular pressure modeled as a Gaussian function.

Note that I have installed FENICS using Docker, and so to run this script I issue the commands:

  1. cd $HOME/fenicsproject/membrane
  2. fenicsproject run
  3. python3 membrane.py
  4. exit
